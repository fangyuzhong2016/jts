Contains the Geometry interface hierarchy and supporting classes.

Java 拓扑套件 （JTS） 是一种 Java API，它使用显式精度模型和可靠的几何算法实现一组核心空间数据操作。 JTS 用于开发支持空间数据集的验证、清理、集成和查询的应用程序。

JTS 尝试尽可能准确地实现 OpenGIS 简单要素规范 （SFS）。 在某些情况下，SFS 不清楚或省略了规范;在这种情况下，JTS 试图选择合理和一致的替代方案。 此规范中记录了与 SFS 的区别和阐述。
