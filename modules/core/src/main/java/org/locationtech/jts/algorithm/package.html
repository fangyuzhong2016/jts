包含的类和接口实现基本的计算几何算法。

稳健性

几何算法涉及组合和数值计算的组合。与使用有限精度数字所有数值计算，选择所述算法易受鲁棒性的问题。 当一个数值计算产生用于由于舍入误差某些输入不正确的回答时，会发生问题的鲁棒性。 健壮性问题在几何运算更为严重，因为它们会导致拓扑构建过程中出现错误。

有许多方法来处理鲁棒性的几何计算的问题。 不足为奇的是，最稳健的算法基本上更加复杂，比非鲁棒版本少高性能的。 幸运的是，JTS是敏感的健壮性仅几个关键功能（如线交点与点在多边形测试）的问题。 有许多可用于这些功能强大的高效的算法，这些算法在JTS实现。

计算性能

运行时性能是一个生产质量实现几何算法的一个重要考虑因素。 在JTS使用的计算量最大的算法是交叉点检测。 JTS方法需要确定以单一几何两个不同几何形状的线段之间的线段（自相交），并且所有的交点之间的所有交叉点。

对交叉点检测（与每个其他每个片段进行比较）明显的幼稚算法具有不可接受的性能降低。 有用于交叉点检测更快的算法大量文献。不幸的是，其中不少涉及大量代码的复杂性。 JTS试图与性能提升，以平衡简化代码。它使用一些简单的技术来生产用于普通类型的输入数据的大幅性能提升。
