包含转换JTS对象和其他格式的接口。

在Java拓扑套件（JTS）是一个Java API实现使用一个明确的精度模型和健壮的几何算法的空间数据的操作的核心组。 JTS意在支持空间数据的验证，清洗，集成和查询应用程序的开发中使用。

JTS试图实现开放GIS简单要素规范（SFS）尽可能准确。 在一些情况下，SFS是不清楚或省略了规范;在这种情况下，JTS试图选择一个合理和一致的解决方案。 从SFS的差异，并阐述都记录在这个规范。
